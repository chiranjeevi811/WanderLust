# WanderLust

A full-stack travel accommodation web application built with Node.js, Express, MongoDB, EJS, Passport.js, and Cloudinary.

## Live Demo

🚀 Deployed Application:
https://wanderlust-1-02qv.onrender.com

## Features

- User authentication
- Create, edit, and delete listings
- Image uploads with Cloudinary
- Reviews and ratings
- Session management
