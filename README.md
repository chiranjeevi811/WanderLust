# WanderLust 🌍

🚧 **Work in Progress** 🚧

WanderLust is a web application to explore, create, and review travel listings. The project is currently under development and features are being added gradually.

## 📌 Current Status
- Basic project structure
- Express.js setup
- MongoDB connection
- Listings CRUD (partial)

## 🛠 Planned Features
- User authentication (login/signup)
- Reviews and ratings
- Image upload
- Authorization & security
- Deployment

## 💻 Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap

## ⚡ How to Run Locally
1. Clone the repository:
```bash
git clone https://github.com/ChiranjeeviAdithya/WanderLust.git
```
2. Navigate into the project folder:
```bash
cd WanderLust
```
3. Install dependencies:
```bash
npm install
```
4. Start the server:
```bash
npm start
```
5. Open your browser at `http://localhost:8080`
